## Endpoint reference

### Core

| API                                                          | Description                                                                                        |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| [Get Action](/api-reference/get-action)                      | Generate a bridge, swap, or cross-chain call transaction.                                          |
| [Get Status](/api-reference/get-status)                      | Track the status of a transaction based on its hash or ID.                                         |
| [Register Transaction](/api-reference/register-transactions) | Trigger indexing for one or multiple broadcasted transactions.                                     |

### Data

| API                                                          | Description                                                                                        |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| [Get Transactions](/api-reference/get-transactions)          | Return transaction history based on desired filters.                                               |

### Utilities

| API                                                          | Description                                                                                        |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| [Get Paths](/api-reference/get-paths)                        | Retrieve available swap routes for given source token based on desired filters.                    |
| [Get Chain List](/api-reference/get-chain-list)              | Return list of all supported chains and their `VmId`s.                                             |

### Workflows | All

| API                                                          | Description                                                                                        |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| [Get Action](/api-reference/get-action)                      | Append a post execution hook to a bridge or swap transaction. **Enables prediction market routes.**                                      |
| [Get User](/api-reference/get-user)                          | Retrieve a Swaps Account for a user. Required for workflows.                                       |
| [Get Workflow Status](/api-reference/get-workflow-status)    | Track the status of a workflow execution.                                                          |

### Workflows | Polymarket

| API                                                          | Description                                                                                        |
| :----------------------------------------------------------- | :------------------------------------------------------------------------------------------------- |
| [Redeem Position](/api-reference/redeem-position)            | Redeem a Polymarket position.                                                                      |
| [Sell](/api-reference/sell)                                  | Sell a Polymarket position.                                                                        |

## Authentication

All API endpoints require authentication using API keys passed via the `x-api-key` header. We have provided a highly ratelimited API key for testing in the API playground.

<Callout type="info">Create a key in the [Console](https://console.swaps.xyz).</Callout>
